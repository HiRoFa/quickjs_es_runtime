fn ignore_list(&self, f: &FileName) -> bool

If true, the file will be in the ignoreList of SourceMap.

Specification for ignoreList: https://tc39.es/ecma426/#json-ignoreList

The ignoreList field is an optional list of indices of files that should be considered third party code, such as framework code or bundler-generated code. This allows developer tools to avoid code that developers likely donโt want to see or step through, without requiring developers to configure this beforehand. It refers to the sources field and lists the indices of all the known third-party sources in the source map. Some browsers may also use the deprecated x_google_ignoreList field if ignoreList is not present.

By default, we ignore anonymous files and internal files.
